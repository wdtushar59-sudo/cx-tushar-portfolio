# fusionai-ai-template

## Author
Tushar Patel

## Description
A dark-themed AI workflow landing page template inspired by the provided screenshot. Built according to the provided HTML Template Portfolio Guidelines: semantic HTML5, BEM CSS, vanilla JS, and a clear folder structure.

This starter includes responsive styles and a mockup to approximate the screenshot's look (dark gradients, glowing accents).

## Folder structure
```
fusionai-ai-template/
├── index.html
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── script.js
│   ├── images/
│   │   └── hero-banner.jpg (placeholder)
│   └── fonts/
└── README.md
```

## How to use
1. Unzip the package.
2. Open `index.html` in your browser.
3. Replace placeholder images in `assets/images/` with optimized assets.
4. Customize colors and spacing in `assets/css/style.css` under `:root` variables.

## Notes
- No external libraries are used (vanilla CSS & JS).
- The layout aims to be a responsive replica focusing on structure and visual language (gradients, glow). For pixel-perfect exactness, replace images and fine-tune spacing/typography.
